Rushk

Rushk, as you probably now if you're on this page, is a Discord bot made for the Koilang server. It's mainly used for storing dictionaries, but is also used for shenanigans. Its uptime is very sporadic and it is available to be brought up at request (ping me).

Style guide

The commands are styled as such

/example-command <required-argument type> <argument-name this|argument|can|be|any|of|these|but|is|required> [optional-argument type] [argument-name this|argument|can|be|any|of|these|but|is|not|required]
The types listed in argument descriptions can be one of the following:

  • number: Any positive number
  • int: Any number, negative or positive
  • text: Any text

In descriptions of commands, if something is styled like this, it is an argument.

Linguistical commands

/dictionary search <dictionary Ciscun|Gedenano|Kadian|Medh|Osrhean|Tsevhu> [specificity Narrow|Broad]
Searches the given dictionary for a term.

Arguments

dictionary: The dictionary to search in
term: The term to search for
specificity: Optional. If set to "narrow", only returns results that are the same as the term; if set to "broad", returns results that contain the term.

/dictionary whatis <dictionary Ciscun|Gedenano|Kadian|Medh|Osrhean|Tsevhu>
Gives an overview of dictionary, including a sample word, the amount of words in the dictionary, and more.

/dictionary random <dictionary Ciscun|Gedenano|Kadian|Medh|Osrhean|Tsevhu>
Gives a random word from dictionary.

WIP /dictionary edit <dictionary Ciscun|Gedenano|Kadian|Medh|Osrhean|Tsevhu> <item Name|Description|Image|Link> <new text>
Allows the manager of a dictionary to edit it, replacing the current item with new

Arguments

dictionary: The dictionary to edit
item: The thing to edit
new: The thing to replace item with
